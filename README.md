I'm **Sergio Shmyhelskyy**, a **Computer Engineering student at [UPC-FIB](https://www.fib.upc.edu/)** (Barcelona) working where **networking, cloud and security intersect**. I design infrastructure that is **reproducible, governed and defensible** — not just configured once and forgotten.

My strongest hands-on work is a **secure hybrid network architecture** (on-prem datacenters + Azure) built for a regulated enterprise: **Terraform** for Infrastructure as Code, **Azure Virtual Network Manager** for centralized SDN governance, **GitOps** change workflows, and a **Zero Trust** access model across dev / pre / prod. That includes operating the **hybrid perimeter** (Check Point + Azure Firewall Premium), **enterprise DNS**, **BGP over ExpressRoute/VPN**, and publishing apps through **Front Door** and Application Gateway.

Outside production-shaped problems, I break things on purpose on **HackTheBox** and **TryHackMe**, experiment with **AI backends in Python**, and run **homelab / IoT** side projects. Currently levelling up **Rust**, **Go**, and **Kubernetes**.
